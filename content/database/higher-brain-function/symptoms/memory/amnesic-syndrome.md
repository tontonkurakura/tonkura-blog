# 健忘症候群

## 説明

記憶だけが選択的に障害され、他の認知機能が比較的保たれている状態を指す。意識は清明で、注意・言語・視空間認知・知能は保たれ、日常会話は成立する。にもかかわらず新しい出来事をほとんど保持できず、しばしば発症以前に遡る逆向性健忘を伴う [1]。「記憶障害がある」だけでは健忘症候群とは呼べず、**他の認知機能が保たれていることの確認が診断の要件**になる。

**選択性を証明する最も強い所見は、手続き記憶が保たれることである。** 重度の健忘患者が、鏡映描写や系列反応時間課題などの技能を練習によって獲得し、成績が日を追って向上していく一方で、その練習を行った事実自体をまったく想起できない、という乖離が繰り返し観察されてきた。これは「記憶」が単一の機能ではなく、意識的に想起される陳述記憶と、行為として発現する非陳述記憶に分かれることを示しており、健忘症候群で障害されているのが前者に限られることの証拠になる。手続き記憶の保存を確認しないまま「全般的な記憶障害」と記載すると、この選択性が記録から失われる。

責任病巣として古くから論じられてきたのが **Papez 回路**と **Yakovlev 回路**である。前者は海馬から脳弓・乳頭体・乳頭視床路・視床前核・帯状回を経て海馬傍回に戻る環状の連絡で、後者は扁桃体から視床背内側核・前頭葉眼窩面／側頭葉前部を経て扁桃体に戻る連絡である。この回路のどこが切れても健忘が生じうるため、健忘症候群は特定の一領域の症候ではなく回路の症候として理解される。原因は多岐にわたり、Korsakoff 症候群、単純ヘルペス脳炎、前交通動脈瘤破裂、低酸素脳症、視床梗塞、脳弓の外科的損傷などが含まれる [2]。

## 病巣

- 内側側頭葉（海馬、海馬傍回、嗅内野）
- 脳弓
- 乳頭体
- 視床背内側核・前核、乳頭視床路
- 前脳基底部（前交通動脈瘤破裂後）
- 上記のいずれか一箇所の両側性病変で成立しうる。一側性病変では軽度にとどまることが多い

## 検査

1. 陳述記憶の評価

   - 言語性素材・視覚性素材の遅延再生と遅延再認
   - 逆向性健忘の範囲（自伝的記憶、公的出来事）

2. 選択性の確認（診断上必須）

   - 注意・作業記憶（数唱、視覚性スパン）
   - 言語（呼称、理解、復唱）
   - 視空間認知・構成
   - 全般的知能

3. 手続き記憶の評価

   - 鏡映描写など、練習による成績向上を日をまたいで確認する課題
   - 課題を行った事実の想起可否と、成績の推移を分けて記録する

4. 標準化検査

   - WMS-R（ウェクスラー記憶検査改訂版）：言語性記憶・視覚性記憶・注意／集中・遅延再生の各指標
   - RBMT（リバーミード行動記憶検査）：日常場面に近い課題での記憶

## 分類上の論点

**「健忘症候群」は病巣を特定する名前ではなく、症候の布置を指す名前である。** 内側側頭葉病変によるものと間脳病変によるものを別の症候群として区別すべきか、同一の症候群の亜型とみなすべきかについては議論が続いている。再認の保たれ方や作話の出やすさに差があるとする報告がある一方 [2]、両者を安定して分ける神経心理学的指標は確立していない。
