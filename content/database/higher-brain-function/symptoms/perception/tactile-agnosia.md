# 触覚失認

## 説明

触覚による物体認知が障害される症候です。要素的な体性感覚（触圧覚、温痛覚、位置覚）は保たれて
いるにもかかわらず、手で探索するだけでは物体が何であるかを認識できません。同じ物体を見せれば
ただちに分かる点で、モダリティに特異的な障害です。

伝統的には二つの水準に分けられてきました。物体の物理的性質を抽出する段階が損なわれる**統覚型**
と、抽出された物体表象を意味に結びつける段階が損なわれる**連合型**です。統覚型はさらに、素材や
材質の弁別が困難になる hyloagnosia と、形や大きさの認知が困難になる morphoagnosia に分けられ
ますが、この両者が乖離した症例はごく少数しか報告されていません。

## 病巣

- 中心後回（一次体性感覚野）— 微小幾何学的（micro-geometrical）情報、すなわち表面の細かな
  凹凸や粗さの知覚に対応する
- 後部頭頂葉・二次体性感覚野（SII）— 病巣がここまで及ぶと巨視的幾何学的
  （macro-geometrical）情報、すなわち長さや形の知覚も障害される
- 頭頂間溝、縁上回、島皮質
- 角回の皮質下白質 — 連合型の責任病巣として報告されている
- 後頭葉（外側後頭側頭領域）— 両側性の連合型触覚失認と両側の皮膚書字覚障害に関与する
  可能性が指摘されている

## 検査

1. 触覚による物体認知

   - 日常物品の触覚的同定（閉眼で手に持たせる）
   - 触覚提示した物品と、視覚提示した選択肢とのマッチング — 呼称の障害と認知の障害を切り分ける
   - 意味のない幾何学的形状の弁別

2. 統覚水準の評価

   - 素材・材質の弁別（hylognosis）
   - 形・大きさの弁別（morphognosis）
   - 微小幾何学的課題（格子の溝の弁別など）と巨視的幾何学的課題（直方体の長さの弁別など）の対比

3. 要素的感覚の評価

   - 触圧覚閾値、温痛覚
   - 二点識別覚
   - 関節位置覚・振動覚

4. 関連する触覚性認知の評価

   - 皮膚書字覚（agraphesthesia の有無）
   - 触覚性の探索運動（能動的探索が可能か）

5. 他モダリティとの対比

   - 視覚提示による同一物品の同定
   - 視覚経由での意味理解 — 意味記憶そのものの障害の除外

## 分類上の論点

**触覚失認と astereognosis（立体覚消失）を同義に扱うことは適切でありません。**

Facci らは、触覚失認が稀でありかつ診断が難しい病態であることを指摘し、その理由として感覚運動
症状がしばしば併存すること、そして**用語法と評価法について合意が存在しないこと**を明記して
います。統覚型を hyloagnosia と morphoagnosia に分ける枠組み自体は提案されているものの、両者の
乖離が示された症例は文献上ごくわずかであり、分類の妥当性はまだ確立していません。

病巣についても、水準ごとに異なる対応が示唆されています。Bohlhalter らは統覚型の2例で、微小
幾何学的特徴の知覚障害が中心後回の病変に、巨視的幾何学的特徴の知覚障害の追加が後部頭頂葉への
病巣の拡大に対応することを示しました。一方、連合型については、角回の皮質下病変によって体性
感覚連合野が下側頭葉の意味記憶から離断されるという離断仮説が提案されています。さらに Sakurai は、
連合型触覚失認・皮膚書字覚障害・運動覚性読字の困難が神経基盤を共有する可能性を SPECT 所見から
論じ、両側性の障害は後頭葉の、一側性ないし一側優位の障害は頭頂葉の損傷に帰されうるとしています。

これに対し **astereognosis という語は、実地の神経学では要素的感覚障害の有無を問わずに使われ
続けています**。報告例を見ると、頸椎症性脊髄症による「numb, clumsy hands」、頸髄髄膜腫、多発性
硬化症の高位頸髄後索病変など、**大脳性でない例が多数を占めます**。これらは後索‐内側毛帯路の
障害による感覚入力そのものの劣化であり、要素的感覚が保たれることを定義に含む触覚失認とは病態が
異なります。Khormi と Atteya は、この混同を避けるために astereognosis と stereoanesthesia
（要素的感覚障害に由来する立体覚の消失）を区別すべきだと整理しています。

用語の使い分けについて分野横断的な合意はまだなく、**文献を読む際にはその論文がどちらの意味で
用語を使っているかを本文で確認する必要があります**。
