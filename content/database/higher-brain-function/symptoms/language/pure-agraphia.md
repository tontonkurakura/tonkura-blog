# 純粋失書

## 説明

書字のみが選択的に障害される症候です。読解や言語理解は保たれていますが、書字の運動プログラミングに障害があり、文字を書くことができません。自発書字、書き取り、写字のいずれも障害されます。

日本語では、漢字と仮名のどちらが侵されるかで責任病巣が異なります。
下の「漢字と仮名の乖離」を参照してください。

## 漢字と仮名の乖離

桜井の総説によれば、**漢字の純粋失書は後部中側頭回の病変**によって生じます。
これは、漢字の**失読失書**を生じる後下側頭皮質（紡錘状回中部・下側頭回、
いわゆる視覚性語形領域）の病変とは区別されます。
すなわち、同じ「漢字が書けない」でも、
読みが保たれているか否かによって示唆される病巣が異なります。

一方、**角回病変による失読失書では漢字の失書**が前景に立ちます。
漢字の書字障害を見たときは、読字が保たれているかどうかを必ず確認してください。

## 病巣

- 後部中側頭回（漢字の純粋失書）
- 中前頭回（Exner領域を含む前頭葉性の純粋失書）
- 上前頭回
- 角回

## 検査

1. WAB失語症検査

   - 書字課題
   - 文字認知課題
   - 言語理解課題

2. SLTA（標準失語症検査）

   - 書字検査
   - 漢字・仮名の処理検査

3. 書字検査

   - 自発書字
   - 書き取り
   - 写字

4. 漢字と仮名を分けた評価

   - 漢字単語・仮名単語の書字
   - 読字が保たれていることの確認（失読失書との鑑別）
