# 失読失書

## 説明

読字と書字の両方に障害を呈する症候。文字の視覚的認知や書字の運動プログラミングの障害により、読解と書字表出の両方に問題が生じる。音読、書き取り、自発書字のいずれも障害される。

日本語では、**漢字（表語文字）と仮名（表音文字）で障害の程度が食い違う**ことがあり、
どちらがどの程度侵されているかによって責任病巣が異なる。
下の「漢字と仮名の乖離」を参照。

## 漢字と仮名の乖離

日本語の失読・失書では、漢字と仮名の解離が中心的な主題であり続けてきた。
桜井の総説は、病巣—症状対応の解析と機能画像の進展にもとづき、
次のような対応を整理している [1]。

- **角回病変による失読失書** — 生じるのは**漢字の失書**である。
  角回病変に伴う仮名の失読は、中後頭回の損傷によるものとされる。
- **後下側頭皮質（紡錘状回中部・下側頭回、いわゆる視覚性語形領域）病変**
  — **漢字の失読失書**を生じる。

したがって、失読失書と一括りにせず、
漢字と仮名それぞれについて読みと書きを評価しないと、病巣の推定は成り立たない。

## 病巣

- 角回（漢字の失書）
- 中後頭回（角回病変に伴う仮名の失読）
- 後下側頭皮質＝紡錘状回中部・下側頭回（漢字の失読失書）
- 縁上回
- 側頭葉

## 検査

1. WAB失語症検査

   - 読解課題
   - 書字課題
   - 文字認知課題

2. SLTA（標準失語症検査）

   - 読字検査
   - 書字検査
   - 漢字・仮名の処理検査

3. 読字・書字検査

   - 単語の読み書き
   - 文章の読解
   - 自発書字

4. 漢字と仮名を分けた評価

   - 漢字単語・仮名単語の音読
   - 漢字単語・仮名単語の書字（自発書字・書き取り）
   - 読みと書きのどちらがより侵されているかの確認
