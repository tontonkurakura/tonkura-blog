# 超皮質性失語

## 説明

復唱は比較的保たれるが、その他の言語機能が障害される失語症です。運動型、感覚型、混合型の3つのタイプがあります。運動型は発話が減少し、感覚型は理解が障害され、混合型は両方の症状を呈します。

## 病巣

- 前頭葉
- 頭頂葉
- 後頭葉

## 検査

1. WAB失語症検査

   - 後頭葉：視覚情報処理に関与（感覚型）

2. SLTA（標準失語症検査）

   - 復唱検査
   - 言語理解検査
   - 発話検査
   - 聴覚的理解の評価
   - 呼称課題

3. SLTA（標準失語症検査）
   - 復唱検査
   - 言語理解検査
   - 発話検査
