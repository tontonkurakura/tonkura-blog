# 語義失語

## 説明

語の**意味**が選択的に失われる失語症候群です。
音は正しく聞き取れているのに、その語が何を指すのか分からなくなります。
1943年、東京大学の精神科医であった**井村恒郎**が新しい失語症候群として提唱し、
「語義失語」と名づけました。

日本語に固有の概念と誤解されることがありますが、そうではありません。
Panse と Shimoyama が1955年にドイツの神経精神医学雑誌で欧米の学界に紹介して以来、
語義失語は音声言語の障害のなかの**独立した症候群として国際的に一定の承認を得てきました**。

山鳥による井村の要約では、語義失語は次の4点で特徴づけられます。

1. 音の知覚は良好であるにもかかわらず、話された語の意味を理解できない
2. 語彙の喪失と語性錯語による表出の障害
3. **復唱の能力は保たれる**
4. 漢字（表語文字）の読み書きが選択的に障害され、仮名（表音文字）の読み書きは保たれる

理解の障害は、症候群の名が示すとおり**名詞などの実質語に限局**しており、
統語の理解は保たれます。
また、保たれた復唱を単なる自動的反応（反響言語）と片づけることはできません。
井村の観察では、患者は理解できない語を拾い上げるような、
問いかけるような態度で復唱していたとされます。

責任病巣について、井村は左側頭葉の第2・第3側頭回あたりと漠然と推定していました。
その後、側頭葉の萎縮を来す変性疾患で語義失語の臨床像が高頻度に見られることが
繰り返し確認されています。

## 超皮質性感覚失語との関係

山鳥は、語義失語の全体像は**大枠では超皮質性感覚失語のカテゴリに収まる**としつつ、
**症候群としての独自性はなお際立つ**と述べています。

違いは規定の粒度にあります。
超皮質性感覚失語は「流暢な感覚性失語で復唱が良好」としか規定せず、
言語のどのレベルが障害されているかを特定しません。
これに対し語義失語は、**障害されるレベルを語（word）に限定**して定義されます。
漢字・仮名の特徴的な解離は、この症候群の診断を助ける所見とされます。

## 病巣

- 左側頭葉（井村は第2・第3側頭回あたりと推定）
- 側頭葉前部・下部の萎縮（変性疾患による場合）
- 病巣局在について、井村の推定を超えて確立された定量的な対応づけは、
  今回の調査では確認できていません

## 検査

1. 標準失語症検査（SLTA）・WAB失語症検査

2. 語の意味の障害を確かめる課題

   - 語の意味の説明（「〜とは何ですか」）
   - 語—絵のマッチング
   - 意味カテゴリ分類
   - 実質語と機能語・統語理解の対比（統語理解が保たれることの確認）

3. 復唱の保存の確認

   - 単語・文の復唱と、その理解との乖離

4. 漢字と仮名の乖離の評価

   - 漢字単語と仮名単語の音読
   - 漢字単語と仮名単語の書字（自発書字・書き取り）

5. 非言語的意味記憶の評価（semantic dementia との関係を検討する場合）

   - 物品の用途の理解、使用動作
   - 絵—絵のマッチング（言語を介さない意味課題）

## 分類上の論点

### semantic dementia との関係

**同一とみる立場**：近年提唱された semantic dementia の概念は、
語の意味の喪失が非言語的な意味記憶の喪失へと症候的に連続することを示唆します。
Sakurai らは、進行性の語義失語で発症した日本人 semantic dementia 患者について、
神経心理学的・画像的・病理学的な検討を行い、
**ユビキチン陽性の前頭側頭葉変性症（FTLD-U）**であったことを報告しました。
また、側頭葉型 Pick 病（英語圏で近年 semantic dementia と呼ばれるもの）が
その初期経過で語義失語の臨床像をしばしば呈することは、
繰り返し確認されています。

**区別を残す立場**：語義失語は**語の意味**の障害として定義された症候群であり、
semantic dementia が要求する**非言語的意味記憶**の障害まで含むかどうかは、
別に検討されるべき問題です。田辺は、側頭葉萎縮による語義失語について、
音韻的・統語的側面が保たれた状態で語の意味記憶が障害される症候群として記述し、
semantic dementia と対比する形でその独自性を論じています。

どちらが正しいかは、ここでは決着させません。

### 漢字・仮名の解離が必須要件かどうか

**未決です。**
山鳥自身が、**井村の記載した漢字・仮名の解離のパターンが症候群の不可欠な一部かどうかは
未解決のままである**と明言しています。
したがって、漢字・仮名の解離を語義失語の**必須所見として扱うべきではありません**。
診断を助ける所見ではありますが、これを欠くことをもって語義失語を否定する根拠にはなりません。

### 変性疾患に限られない

語義失語ないしそれに類する臨床像は、変性疾患の専有ではありません。

- **ヘルペス脳炎**：単純ヘルペス脳炎が疑われた症例で、特異な情動・行動障害とともに、
  新造語および新造語的な漢字処理を伴う語義失語様症候群が報告されています。
- **筋萎縮性側索硬化症（ALS）**：漢字の失読と語義失語を呈した症例が報告されています。

血管障害による報告もありますが、
今回の調査ではその一次文献を確認できていません（未確認）。
