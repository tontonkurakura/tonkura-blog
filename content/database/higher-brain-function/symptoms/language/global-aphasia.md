# 全失語

## 説明

発話・理解・復唱・呼称のすべての言語モダリティが重度に障害される、最も重い型の失語症です。
発話はまったく出ないか、意味をなさない常同的な発話（再帰性発話）に限られ、
簡単な指示の理解も困難です。

**連続例の調査では最も頻度が高い型**であり、初発脳卒中による失語60例の検討では
33.3%を占めました。

通常は右片麻痺を伴いますが、**片麻痺を伴わない全失語**という亜型が知られています。
これは左前頭葉と側頭葉に離れた二つの病変があり、あいだの中心前回が保たれる場合に生じ、
塞栓症や硬膜動静脈瘻が原因となります。片麻痺がないために失語の重症度が
見落とされやすく、注意を要します。

## 病巣

- 左中大脳動脈本幹領域（前頭・側頭・頭頂の広範な損傷）
- 片麻痺を伴わない全失語では、左前頭葉と側頭葉の**二連続病変**（中心前回を回避）

## 検査

1. 標準失語症検査（SLTA）

2. WAB失語症検査

3. 重度例向けの評価

   - 残存するコミュニケーション手段の確認（ジェスチャー、表情、うなずき）
   - 自動言語（数唱、曜日）の確認
