# 視覚失語

## 説明

視覚的に提示された物品の**呼称だけ**が障害される症候です。
同じ物を手に触らせれば言えますし、音を聞かせれば言えます。
障害は視覚というモダリティに限られます。

視覚失語の要点は、**意味へのアクセスが保たれている**ことです。
名前は出てこないのに、その物が何に使うものかを説明でき、使い方をジェスチャーで示せます。
つまり物の同定そのものは成立しており、視覚情報から語形へ至る経路だけが断たれています。
この点が、物の同定自体が成立しない**連合型視覚失認**との決定的な違いです。
両者はいずれも「視覚提示物の呼称障害」として現れるため、
意味へのアクセスが保たれているかどうかを確かめないと区別できません。

機序は、右半球の後頭葉における視覚処理と、左半球の言語野との離断と考えられています。
左後頭葉病変と脳梁膨大部（splenium）の障害の組み合わせで生じ、
右同名半盲を伴うことが典型です。

Veronelli らは、左後頭葉病変と脳梁膨大部経路の損傷を有する81歳女性例を報告しています。
複数の視覚的物体処理課題を用いた検討で、行動データからは
**視覚入力からの意味アクセスが可能である**ことが示され、
病巣にもとづく structural disconnectome 解析では膨大部の関与が示されました。
著者らは、視覚—言語間の脳梁離断という想定が支持されると結論しています。

## 病巣

- 左後頭葉
- 脳梁膨大部（splenium）およびそこを通る半球間経路
- 左後大脳動脈領域の梗塞で生じることが多い
- 右同名半盲をしばしば伴う

## 検査

1. 呼称のモダリティ間比較（診断の中核）

   - 視覚提示による呼称
   - 触覚提示（閉眼で手に持たせる）による呼称
   - 聴覚提示（物の音）による呼称
   - 語による定義からの呼称

2. 意味アクセスの保存を確かめる課題

   - 用途の口頭説明
   - 使用動作のジェスチャーによる提示
   - 意味カテゴリ分類・同カテゴリ内での物品のマッチング

3. 視覚認知そのものの評価

   - 図形の異同弁別、模写
   - 視野検査（同名半盲の有無）

4. 標準失語症検査（SLTA）・WAB失語症検査（全般的な言語機能の確認）

## 分類上の論点

視覚失語と連合型視覚失認を独立した2つの症候とみなすか、
視覚から意味・語形へ至る処理の連続的な障害の程度の違いとみなすかについては、
議論があります。実際の症例では意味アクセスの保存が部分的なことがあり、
両者の境界が常に明瞭とはかぎりません。
本項で挙げた報告は単一症例研究であり、
この境界に関する一般的な結論を与えるものではありません。
