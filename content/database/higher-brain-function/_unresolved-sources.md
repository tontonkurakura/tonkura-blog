## 4. 事実として確認できなかったこと

推測で埋めていない箇所です。データベース本文にも「未確認」と明記してあります。

1. **失音楽の「感覚性／運動性」の対比を明示的に定義した文献。**
   現代の総説（PMID 35964992）は先天性／後天性の軸で整理しています。
   現行の項目名を「感覚性失音楽」から「失音楽」に変更したのはこのためです。
   日本語の教科書的記載を別途あたる必要があります。
2. **SLTA・WAB 日本語版における失語型の分類不能例の割合。**
   古典分類に収まらない症例が一定数あることは広く知られていますが、
   日本語の検査での具体的な数値を確認できませんでした。
3. **DSM-5 本文の6神経認知領域の正確な記述。**
   「複雑性注意」に半側空間無視が含まれるか否かを原典で確認できていません。
   本データベースでの DSM-5 への言及は、枠組みとしての参照にとどめています。
4. **大脳性色覚障害の下位分類**（achromatopsia と color agnosia / color anomia の区別）。
   色覚障害そのものの下位分類を扱う文献を取得していません。
5. **展望記憶障害・手続き記憶障害・既視感／未視感の病巣。**
   症候としての存在は標準的ですが、個別に文献検証したものではなく教科書的知識です。
