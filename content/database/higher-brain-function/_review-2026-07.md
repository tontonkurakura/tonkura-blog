# 高次脳機能データベース 症候リストの検討

`_` 始まりの作業文書。一覧には出ない。2026-07-19 作成。

現行の6カテゴリ50項目（実ファイル51本）について、症候の過不足と分類の妥当性を
文献にあたって検討した記録。**分類軸の最終決定は本人が行う。**ここは判断材料を出すところまで。

---

## 収載範囲

**高次脳機能障害の全般**を扱う。DSM-5 の6神経認知領域（複雑性注意／遂行機能／学習と記憶／
言語／知覚–運動／社会的認知）＋半球離断症候群を骨格にする。

この骨格は永井知代子の総説（神経治療 39: 241–245, 2022）が採る整理で、
勉強会の守備範囲（失語・失行・失認・記憶・注意・遂行機能）とも一致する。

### 学術的な定義と行政的な定義は別物

厚生労働省の診断基準（2006年、および令和5年版ガイドライン）は冒頭で自ら区別している。

> 「高次脳機能障害」という用語は、学術用語としては、脳損傷に起因する認知障害全般を指し、
> この中にはいわゆる巣症状としての失語・失行・失認のほか記憶障害、注意障害、遂行機能障害、
> 社会的行動障害などが含まれる。

**行政的定義はこのうち記憶・注意・遂行機能・社会的行動障害の4つに限定**され、失語・失行・失認を
含まない。原因疾患も外傷・脳血管障害・低酸素脳症・脳炎・脳腫瘍に限られ、Alzheimer病等の
進行性疾患を除外する。

このデータベースは**学術的定義**に立つ。ただし行政的定義は臨床で必ず出会うので、
どこかに明示的な対照表を置くべきである（後述）。

### 令和5年版ガイドラインは半側空間無視を「注意障害」に置いている

原文の構成は次のとおり。

```
2.注意障害
  ①全般性注意障害（集中困難・注意散漫／注意の持続・維持困難）
  ②半側空間無視
```

軽度の記述に「両側同時刺激を行うと病巣反対側を見落とす、すなわち一側消去現象(extinction)」
とある。**現行リストが半側空間無視を「認知の障害」に置いているのは、この建て付けと合わない。**

### 分類軸が混在している

記憶／言語／行為／認知は機能軸だが、**前頭葉だけが部位軸**。
DSM-5 の枠に写すと、前頭葉カテゴリの中身は次のように分かれる。

| 現行「前頭葉の障害」の項目           | 帰属先                                                                 |
| ------------------------------------ | ---------------------------------------------------------------------- |
| 遂行機能障害                         | 遂行機能                                                               |
| 意欲障害（アパシー）                 | 遂行機能／動機づけ（**論争あり**、後述）                               |
| 脱抑制・感情失禁・性格変化           | 社会的認知／前頭葉性脱抑制                                             |
| 保続                                 | 遂行機能・言語・行政の社会的行動障害に跨る（**論争あり**）             |
| 道具の強迫的使用・模倣行動・把握現象 | 環境依存症候群（知覚–運動／遂行機能の境界）                            |
| 拮抗失行                             | **半球離断**（現行は「その他」の脳梁離断症候群と分離してしまっている） |

### 階層の取り違え

**症候・症候群・疾患が同列に並んでいる。**

- **Anton症候群は病態失認の下位型である。** Anton (1899) が皮質盲・皮質聾の障害無自覚を概念化し、
  Babinski (1914) が**その臨床実体を指すために anosognosia を造語**した（PMID 31220837,
  [DOI](https://doi.org/10.1159/000494954)）。両者を同列に置くのは論理的に整合しない。
- **Balint症候群は症候群**で、3徴（同時失認・視覚性運動失調・精神性注視麻痺）はいずれも
  現行リストに無い。一側病変で非対称な不全型が出るため、症候群名だけでは記録できない
  （PMID 29199507, [DOI](https://doi.org/10.1080/13546805.2017.1407304)）。
- **Gerstmann症候群が立項されていないのに、構成要素の失計算と左右盲だけがある。**
  残る2つ（**手指失認・失書**）が欠けている。Gerstmann 自身は手指失認を中核とみなしていた
  （PMID 35998548, [DOI](https://doi.org/10.1016/j.cortex.2022.07.002)）。
- **「アルツハイマー型認知症による記憶障害」は疾患名を冠した項目。**同じ論理なら
  Lewy小体型・血管性・外傷性…と疾患数×症候数に発散する。症候記述（内側側頭葉型健忘）に
  置き換え、疾患側からリンクするのが一貫する。

### 表記のゆれ

`失名辞失語` / `失名詞失語`、`失計算` / `失算`、`触覚性失認` / `触覚失認`。
リポジトリ内にもう1本の木（`content/neurology/Neuroanatomy/高次脳機能/`）があり、そちらと不一致。

> なお **「左右盲」より「左右失認」** が文献の用語（right-left disorientation）に近い。

## 領域別の過不足

### 複雑性注意 — カテゴリごと存在しない

**最も大きな欠落。** 現行に注意のカテゴリが無く、CAT（標準注意検査法）の言及も51本中ゼロ。

| 追加すべき項目                           | 英語名                                                | 病巣                                 | 検査                                 |
| ---------------------------------------- | ----------------------------------------------------- | ------------------------------------ | ------------------------------------ |
| 全般性注意障害（覚度・持続・選択・分配） | vigilance / sustained / selective / divided attention | 上行性網様体賦活系、視床、右前頭頭頂 | CPT、抹消課題、Stroop、PASAT         |
| 消去現象                                 | extinction                                            | 頭頂葉                               | 両側同時刺激（視覚・触覚・聴覚）     |
| 運動無視                                 | motor neglect                                         | 前頭・頭頂・視床                     | 行動観察、placing reaction           |
| 方向性運動低下                           | directional hypokinesia                               | 前頭葉優位                           | 視野と運動方向を分離した反応時間課題 |
| 表象無視                                 | representational neglect                              | 右半球                               | ミラノ大聖堂課題、表象二等分課題     |

運動無視の原著は Laplane & Degos の20例（PMID 6842219,
[DOI](https://doi.org/10.1136/jnnp.46.2.152)）。表象無視は Bisiach & Luzzatti
（PMID 16295118, [DOI](<https://doi.org/10.1016/s0010-9452(78)80016-1>)）。

**注意の4分法（持続/選択/転換/分配）の原典（Sohlberg & Mateer）は PubMed では同定できなかった。**
教科書由来のため。**未確認**。

### 学習と記憶 — 4項目は薄い

現行4項目は軸も混在している（疾患名1・発作性健忘2・産出症状1）。

| 追加すべき項目                                       | 補足                                          |
| ---------------------------------------------------- | --------------------------------------------- |
| 前向性健忘 / 逆向性健忘                              | 時間的勾配（Ribot 則）                        |
| 健忘症候群                                           | Papez回路／Yakovlev回路                       |
| **Korsakoff症候群**                                  | 乳頭体・視床背内側核。確立した固有名の症候群  |
| **一過性全健忘 TGA**                                 | **TEA だけあって TGA が無いのは鑑別上不自然** |
| 加速的長期忘却                                       | 標準検査で捉えられない。TEA で顕著            |
| 意味記憶障害／手続き記憶／展望記憶／ワーキングメモリ |                                               |

TGA について実務上重要な2点。急性期には時空間失見当・意味記憶検索困難・
ワーキングメモリ障害を伴い、**24時間後も微細な再認障害が残りうる**（PMID 39090229,
[DOI](https://doi.org/10.1007/s00415-024-12563-2)）。DWI の感度は時間依存で、
**発症0–12時間では15.6%**、48–60時間で82.8%（PMID 34516708,
[DOI](https://doi.org/10.1111/acem.14390)）。急性期の陰性MRIで否定できない。

**作話は自発性と誘発性に分けるべき。** 自発性作話25病巣の lesion network mapping で
100%が乳頭体と機能的結合を持ち、健忘の病巣より眼窩前頭皮質との結合が強い
（PMID 37415502, [DOI](https://doi.org/10.1176/appi.neuropsych.20220160)）。

|            | 道具の強迫的使用                                                                     | 使用行動                                                                     |
| ---------- | ------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------- |
| 提唱       | 森悦朗・山鳥重 1982（PMID 7127975）                                                  | Lhermitte 1986（PMID 3707084, [DOI](https://doi.org/10.1002/ana.410190404)） |

### 遂行機能・社会的認知

遂行機能の因子構造は Miyake の unity/diversity（shifting / updating / inhibition）が標準
（PMID 10945922, [DOI](https://doi.org/10.1006/cogp.1999.0734)）。

社会的認知として立てるべきは、心の理論、表情認知障害、共感の障害、失感情症、
認知コミュニケーション障害。ただし**「共感」と「心の理論」を並列に立てると二重計上になる**。
Schurz らは両者を「収束の限られた異種過程を覆う傘概念」と呼び、概念と分類法の合意が
欠如していると明言している（PMID 33151703, [DOI](https://doi.org/10.1037/bul0000303)）。
**認知的共感（≒心の理論）と情動的共感**の軸で分けるのが安全。

**行政の「社会的行動障害」5項目は、DSM-5 の「社会的認知」と1対1対応しない。**

| 行政カテゴリ            | 実際の帰属                     |
| ----------------------- | ------------------------------ |
| ①意欲・発動性の低下     | アパシー → 遂行機能／動機づけ  |
| ②情動コントロールの障害 | 前頭葉性脱抑制（眼窩前頭）     |
| ③対人関係の障害         | **ここだけが社会的認知に対応** |
| ④依存的行動             | 発動性低下＋人格変化の複合     |
| ⑤固執                   | 保続／set-shifting → 遂行機能  |

村井俊哉は、社会的行動障害を「特定の脳領域との明確な対応関係があるものではなく、
さまざまな問題行動の総称」「概念を規定する背景理論が希薄」とし、**遂行機能障害・アパシー・
脱抑制の3症候群への再分類**（DLPFC / MPFC / OFC 対応）を提案している
（Jpn J Rehabil Med 2018;55:46-51）。ただし村井自身が
「損傷部位から行動変化のパターンを予測するところまではいかないことがほとんど」と留保している。

### 半球離断

現行は「その他」に脳梁離断症候群1本のみ。**拮抗失行（前頭葉カテゴリ）はここに属する。**
永井の総説も半球離断を独立領域として扱っている。

## 分類が定まっていない論点

**単一の正解として書けない。両論併記が要る。**

1. **半側空間無視を注意に置くか視空間認知に置くか。**
   注意説＝Corbetta & Shulman の注意ネットワーク仮説（PMID 21692662,
   [DOI](https://doi.org/10.1146/annurev-neuro-061010-113731)）、行政ガイドライン。
   表象説＝Bisiach の心的イメージ課題、石合の表象二等分課題（PMID 11080226,
   [DOI](https://doi.org/10.1136/jnnp.69.6.745)）。
   第三の立場＝多成分症候群説（知覚–感覚性と探索–運動性の二重乖離、PMID 1527542,
   [DOI](https://doi.org/10.1136/jnnp.55.8.701)）。
   **主収載は「複雑性注意」、「知覚–運動」から相互参照**を推奨する。
2. **視覚失認の分類粒度。** 2分類も3分類も粗すぎるという批判（PMID 12916489,
   [DOI](<https://doi.org/10.1016/s0733-8619(02)00095-6>)）。
3. **相貌失認の右前部側頭葉型。** 連合型相貌失認か多感覚的人物認知障害か。
   Gainotti（PMID 23579426, [DOI](https://doi.org/10.1007/s11065-013-9232-7)）と
   Barton & Corrow（PMID 26375683, [DOI](https://doi.org/10.1016/j.neuropsychologia.2015.09.015)）が対立。
4. **Gerstmann症候群の単一性と局在。** Cubelli & Rusconi は原著3症例を再検討し
   「共通の機能的分母は現れてこない」と結論（PMID 35998548）。一方で純粋型の報告は続いている。
   局在も左角回以外（左下前頭回、右半球）の報告がある。
5. **病態失認と疾病否認。** 認知要因か動機づけ要因か。左半球病変例も稀ではない
   （PMID 41234934, [DOI](https://doi.org/10.3389/fneur.2025.1681303)）。
6. **触覚失認と astereognosis。** 用語法に合意が無い（PMID 39118817,
   [DOI](https://doi.org/10.3389/fnhum.2024.1401578)）。astereognosis は脊髄病変でも生じる
   記述的用語。**同義扱いしない。**
7. **構成障害・着衣失行が単一の障害か。** 構成障害は「均質な構成概念ではない」
   （PMID 29519467, [DOI](https://doi.org/10.1016/B978-0-444-63622-5.00016-4)）。
   着衣失行は右半球例では視空間障害・注意低下、左半球例では観念運動失行により生じ、
   着衣固有の機序は同定されていない（PMID 17127572, [DOI](https://doi.org/10.1080/09602010500204385)）。
8. **アパシーの帰属。** 遂行機能の下位／社会的行動障害／独立した動機づけ障害の3通り。
   Robert 基準・ISCTM 基準は独立の臨床構成概念として定義している（PMID 33949763,
   [DOI](https://doi.org/10.1002/alz.12358)）。
9. **保続の帰属。** 再帰性保続は失語（後方病巣）と本質的に結びつき、右半球病巣では
   持続性保続が主（PMID 2423294, [DOI](<https://doi.org/10.1016/s0010-9452(86)80035-1>)）。
   **「保続＝前頭葉性遂行機能障害」ではない。**
10. **失語型分類そのもの。** 初発脳卒中失語60例で、伝統的分類による病巣と型の対応が
    明確だったのは**35%のみ**。型別内訳は全失語33.3%、Broca 28.3%、超皮質性運動13.3%、
    超皮質性感覚10%、Wernicke 8.3%、失名辞5%、伝導1.7%（PMID 26713015,
    [DOI](https://doi.org/10.4103/0972-2327.165469)）。
    **現行が伝導失語を立てて全失語を立てていないのは頻度の実態と逆行している。**

## 収載基準の案

1. **症候テーブルと疾患テーブルを分ける。**
   PPA の3亜型・PPAOS、「AD による記憶障害」は疾患側に置き、構成症候へリンクする。
   PPA は症候群記述から病因診断へ移るべきという議論があり（PMID 28515274,
   [DOI](https://doi.org/10.1212/WNL.0000000000004041)）、剖検確認例のクラスタ解析でも
   頑健な下位群は2つしか取れていない（PMID 37988272, [DOI](https://doi.org/10.1093/brain/awad396)）。
2. **症候群には構成症候への親子関係を持たせる。**
   Balint、Gerstmann、Korsakoff、Anton、半球離断。
3. **論争フラグを frontmatter に持つ。** 上記10項目に該当するものは本文で両論を書く。
4. **Papez回路・Yakovlev回路は症候ではない。** 病巣メタデータとして持つ。
5. **臨床評価尺度の項目文は書かない。** 検査名と何を測るかに留める（WAB・SLTA・CAT・VPTA・
   BIT・RBMT・BADS 等は著作物）。
